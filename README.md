# Greystone Labs Loan API

## Setup

1. Create and activate a virtualenv

```
virtualenv someEnv
source someEnv/bin/activate
```

2. Install requirements

```
pip install -r requirements.txt
```

3. Run. Navigate to provided URL.

```
uvicorn main:app --reload
```

## Testing

Simply run
```
python3 tests.py
```
